Model Evaluation

In[45]:
# Train & evaluate each feature set

for name, (X_features, feature_list) in feature_sets.items():
    X_train_fs = X_features.iloc[indices_train]
    X_val_fs = X_features.iloc[indices_val]
    X_test_fs = X_features.iloc[indices_test]

    y_train_fs = y.iloc[indices_train]
    y_val_fs = y.iloc[indices_val]
    y_test_fs = y.iloc[indices_test]

    # Create pipeline

    pipeline = Pipeline([
        ('scaler', StandardScaler()),
        ('model', LinearRegression())
    ])

    # Train on training set

    pipeline.fit(X_train_fs, y_train_fs)

    # Evaluate on training set

    y_train_pred = pipeline.predict(X_train_fs)
    train_rmse = np.sqrt(mean_squared_error(y_train_fs, y_train_pred))
    train_r2 = r2_score(y_train_fs, y_train_pred)

    # Evaluate on validation set

    y_val_pred = pipeline.predict(X_val_fs)
    val_rmse = np.sqrt(mean_squared_error(y_val_fs, y_val_pred))
    val_r2 = r2_score(y_val_fs, y_val_pred)

    # Record results

    results.append({
        'Feature Set': name,
        'Number of Features': len(feature_list),
        'Train RMSE': train_rmse,
        'Val RMSE': val_rmse,
        'Train R^2': train_r2,
        'Val R^2': val_r2,
        'Generalization Gap': val_rmse - train_rmse
    })

Ethical Concerns & Model Limitations

Ethical Concerns

  • Could lead to discriminatory practices if the model is implemented (larger cost differences for "smokers" & "obese")
  • Model uses senstive information (smoking, BMI, etc.) which needs to be handled carefully for individuals' privacy.
  • Bias in the model can lead to unfair treatment (price differences based on region, sex, etc.)

Data & Model Limitations

  • Feature set is fairly limited; could be other relevant factors like chronic conditions, lifestyle, medication usage, etc.
  • Dataset could not be representative of all racial/ethnic groups; can perfrom poorly for under-represented populations
  • RMSE and R-squared show good model performance overall, but the model could underperform for subgroups (age, region, etc.)

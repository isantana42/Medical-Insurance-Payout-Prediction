# Medical-Insurance-Payout-Prediction

## Background
The objective of this project is to develop a predictive model through multple linear regression. The project aims to provide insights into which features are most important in predicting payout, how regional differences affect customer charges, and common characteristics among higher payouts. Answering these questions can help insurance specialists simplify the enrollment process for new customers and flag high-risk customers. 

## Dataset Info:

1. age - age of the customer
2. sex - gender of the customer
3. bmi - body mass index
4. children - number of children
5. smoker - whether the customer smokes or not
6. region - which region of the country the customer belongs to
7. charges - target variable; expenditure of the customer
